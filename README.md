# LiMiTools-for-Unity
LiMiTools for Unity是LiMi开发的第一个Unity编辑器工具包

## 使用方法：打开Unity -> 窗口 -> 包管理器 -> “+” -> 安装来自git URL 的包...  -> 输入(https://github.com/LiMiSensei/LiMiTools-for-Unity.git)

## Prerequisites
